Counting in bin and hex

The number of digits used in a numbering system is called number base or radix.

Positional Notation

Our number systems are positional, because each digit represents a value that is dependent of its position.

1023 and 3102 contain the same digits, but their position give them different "values".

The digit at the far left is the most significant, the one at the far right is the least significant (MSB/LSB for binary system).

Math says: if we have a number formed by digits dm...d2d1d0 in a given number base n, the number can be interpreted as:

dm * nm + ... + d2 * n2 + d1 * n1 + d0 * n0

IPv4: Bin to Dotted Decimal

  1. Group the 32 bits into 4 octects.
  2. Convert each octet to decimal.
  3. Add a '.' (dot) between each decimal.

IPv4: Dotted Decimal to Bin

Baseexp 27 26 25 24 23 22 21 20
Weight 128 64 32 16 8 4 2 1
Bin ? ? ? ? ? ? ? ?
  1. Is the decimal number > or = than the MSB (128)?
    • If yes, put "1" in the MSB and subtract it from the number.
    • If no, put "0" in the MSB.
  2. Move to the next (less significant) position and repeat with the remainder.
  3. Repeat until you have exhausted all bit positions.
  4. Repeat for all 4 decimals.

Hexadecimal Numbering System

Base-16 is a perfect number system to use in technology because any of the 0-F digits represent a group of 4 bits.

This means that a common grouping such as 1 byte (8 bits) can be represented in hex with the range 00-FF.

Hex numbers are usually represented by preceding the value with "0x". For instance, the number 7A (122 in decimal) is represented in hex with 0x7A, 7AH, or 7A16.

Hex Conversions

  • Hex to Binary: just convert each hex digit into its binary 4-bits values and write them one after the other.
  • Binary to Hex: split the binary number into 4-bits groups and convert them into their hex values, and write them one after the other.
  • Hex to Decimal: use the positional system or convert first to binary and then to decimal.
  • Decimal to Hex: use the repeated divisions-by-16 method (write down the remainders in hex digits, e.g. "C" for a remainder of 12), or convert first to binary and then to Hex.
